What is this website?

This website was made so that it would recommend parts to people who do not know too much about computers. This is a Work-In-Progress (WIP) and I plan to add many more features to help newer PC builders! Currently this site only has a Pc part recommender. You can choose your budget, your choice of perfomance, choice of type of CPU (Intel or AMD), and choice of GPU (Geforce or Radeon).

User inputs

Budget

This is the total budget you are willing to spend on a computer build. Note this does not include peripherals and is only for the actual build itself. Note: Minimum £600.


Perfomance

This option is here so you can customise if you want your CPU or CPU to be more powerful. For most people, it is recommended to choose a balanced setup. However, A more powerful GPU allows for more graphically challenging games to be run at higher frames, but if the application is CPU demanding, then it may run slower. If you want to edit videos and pictures only, a CPU intensive setup is recommended. Choose wisely, as each intensive option reduces budget for the other important part.


Central Processing Unit - CPU

The remaining options are simply preference. The newest generation of each brand is quite even for the price/perfomance ratio, so perfomance isn't being lost by choosing the other brand. The central processing unit is used for calulcating everything your computer does, and handles all the information.




Graphics Card - GPU

In graphics cards, the price/perfomance ratio is still quite even. Geforce is currently more popular due to more usability features, since they have been dominating the market for longer. Radeon is the AMD competitor which recently improved their graphics cards massively in order to compete, but it is still missing a few quality of life features that geforce already has.
*Warning: Current GPU prices are still quite high due to the COVID pandemic.*
